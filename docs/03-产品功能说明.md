# 慢慢说｜产品功能说明

## 产品定位

慢慢说是一款面向老年人与异地子女的 AI 生活翻译器，核心能力是把复杂数字服务翻译成老人能理解、能行动、能求助的表达。

## 核心原则

- 不责备：不说“你操作错了”，而是说“没关系，我们一步一步来”
- 不恐吓：遇到风险时，先告诉老人“先不要点”，再解释原因
- 不替代：AI 不替老人做关键决定，而是帮助老人理解和确认
- 不堆功能：每次只讲眼前这一步，避免信息过载
- 不忽视旧手机：默认考虑低端机、旧系统、小屏幕和弱网环境
- 不要求一次说清：允许老人表达零散、前后不一致或记不清
- 主动多想一步：不只回答当前问题，还要识别老人可能没有意识到的风险

## 页面结构

### 首页入口

当前 Demo 有三个入口：

- 我不会用手机
- 这是不是诈骗
- 帮我跟家人说

### 识别与输入区

老人可以：

- 上传截图
- 直接使用示例
- 输入一句自己的问题

### 结果区

结果固定遵循五段结构：

1. 安抚：先别急，我们一起看
2. 结论：是否有风险 / 下一步是什么 / 已经整理好
3. 解释：用老人能理解的话说明原因
4. 行动：给出最多三步建议
5. 求助：生成给家人的清楚消息

## 三个 Demo 场景

### 场景一：这是不是诈骗

输入：医保异常短信  
输出：

- 可能有风险
- 不要点陌生链接
- 不要输入身份证、银行卡或验证码
- 生成给家人的求助消息

### 场景二：我不会用手机

输入：医保电子凭证登录页面  
输出：

- 先确认手机号
- 再获取验证码
- 验证码只填在当前医保页面里
- 生成给家人的确认消息

### 场景三：帮我跟家人说

输入：这个医保又弄不好了  
输出：

- 将模糊表达整理成清楚问题
- 告诉家人老人卡在医保操作中
- 提醒在确认前不要反复尝试

## 路演模式

右上角「路演模式」会自动播放三幕故事：

1. 先保护老人
2. 把操作讲慢
3. 让求助变清楚

这个模式用于录屏、现场演示和评审快速理解产品闭环。

## 2.0 家庭共护

- 用高风险、需确认、暂未发现风险三级表达当前判断
- 告诉老人判断依据，不只给出结论
- 对“已经点过链接”等情况先安抚，再给紧急处理步骤
- 将老人的问题、AI 判断和行动建议组成家庭求助卡
- Demo 可演示家属收到问题并回复“先不要操作，我来帮你确认”
